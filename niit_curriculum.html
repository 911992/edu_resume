DNIIT Certificate Curriculum

DNIIT was a quarter based software engineering certificate. For each quarter students were asked to prepare and present a generic-tech topic(aka ISAS), same based on top subjects (per quarter), 2 or 3 projects were assigned to group of students to analyze, design and implement them.

  • ICDL and Fundamental of Programming (Q1)

    MS Office(word, excel, powerpoint, access), and basic algorithms using flowchart and pseudocode

  • Networking And Operating Systems Essentials (Q2)

    Managing and administrating computer networks(TCP/IP).

  • Design and Implement Relational DB (Q3)

    Data analyzing , normilizing, and designing using RDBM systems and SQL(MS SQL)

  • Web Development and Object Oriented Programming Using Java SE (Q4)

    Static web development (using HTML, CSS, and JavaScript). Design and implementing object oriented based applications using Java tech.

  • Fundamental of Software Engineering and Linux OS (Q5)

    Administrating and scripting using bash on Linux systems. SDLC.

  • Software Designing and Data Interchange Language (Q6)

    Design and engineering object oriented applications using UML and design patterns. Manipulating XML files for interchanging(and presenting) data between heterogeneous systems.

  • Enterprise Wide Applications Development using JEE (Q7)

    Design, and implementing web(full-stack) applications using JEE web-profile.

  • Enterprise Wide Solution Architect using JEE (Q8)

    Design, and implementing enterprise, message oriented, and n-tier systems using JEE and EJB technologies. Website design methodology(UI/UX)
